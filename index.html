Setup guide

Up and running
in under 5 minutes.

Four steps. No terminal. No technical knowledge required.

1
Download the installer
Click the Download button and save Zerqon.Setup.exe to your PC.
2
Run the installer
Double-click the file. Click Yes on the Windows popup. Installs in under 30 seconds.
3
Enter your API key
The setup wizard opens automatically. Paste your free Gemini key when asked.
4
Join a meeting
Open any meeting app. Zerqon detects it and asks if you want AI assistance.
Getting your API key

Free Gemini key
in 60 seconds.

Zerqon uses Google's Gemini AI. The free tier gives 1,500 requests per day — more than enough for meetings.

1
Go to Google AI Studio
Open aistudio.google.com/app/apikey in your browser. Sign in with your Google account.
2
Create a new API key
Click Create API key, select any project, and confirm. Your key will appear instantly.
3
Copy the key
Click the copy icon (📋) next to your key. It starts with AIzaSy and is 39 characters long. Do not share this key with anyone.
4
Paste it into Zerqon
During the setup wizard (Step 2), paste the key into the API key field. Zerqon saves it securely on your PC — it never leaves your machine.
5
Change it later if needed
Right-click the Zerqon tray icon → Settings → API Keys tab → paste a new key anytime.
